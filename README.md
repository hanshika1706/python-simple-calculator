# Python Simple Calculator

This is a simple command-line calculator built using Python.  
It allows users to perform basic arithmetic operations by entering two numbers and selecting an operation.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

## Technologies Used
- Python

## How to Run

1. Clone the repository
git clone https://github.com/your-username/python-simple-calculator.git

2. Navigate to the project folder
cd python-simple-calculator

3. Run the program
python calculator.py

## Example

Enter first number: 10  
Enter second number: 5  

Choose operation:  
1. Addition  
2. Subtraction  
3. Multiplication  
4. Division  

Enter choice (1/2/3/4): 1  

Result: 15
